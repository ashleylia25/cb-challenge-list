# CB Challenge List
*Idea proposed by Kepler*

This website is meant for keeping track of all the Geometry Dash challenge levels created by people in CBCL.

## Includes:
1. A list of all the challenges ranked in order from hardest to easiest. (Has a search bar)
2. Completions submitted by the players.
3. Stats viever - Sort of a leaderboard where players gather points and climb up the rankings with each completed challenge (the ammount of points one gains for each completion is based on the difficulty of said challenge)
